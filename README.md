# Blog Management API V2

A RESTful Blog Management API built using Spring Boot and PostgreSQL.

## Features
- Create Blog Post
- Get Blog Post By ID
- Get All Blog Posts
- Update Blog Post
- Delete Blog Post
- Validation
- Exception Handling
- Custom Error Responses

## Technologies Used
- Java 17
- Spring Boot
- Spring Data JPA
- PostgreSQL
- Maven
- Postman

## API Endpoints

POST    /api/posts
GET     /api/posts
GET     /api/posts/{id}
PUT     /api/posts/{id}
DELETE  /api/posts/{id}
